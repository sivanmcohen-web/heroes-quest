# 🏆 משימות הגיבורים - PWA
## מערכת משפחתית לניהול משימות ולמידה

---

## 📦 מה יש בתיקייה?

כל הקבצים שצריך כדי להפוך את המערכת ל-PWA (Progressive Web App):

- `index.html` - דף הבית עם ניווט לכל האפליקציות
- `signatures.html` - מערכת כרטיסיות החתימות
- `games.html` - משחקי הלמידה האינטראקטיביים
- `reports.html` - דוחות והתקדמות
- `manifest.json` - הגדרות PWA
- `sw.js` - Service Worker (לעבודה offline)
- `icon-192.png` ו-`icon-512.png` - אייקונים (תצטרכי ליצור)

## 🚀 איך להעלות? (3 אופציות)

### **אופציה 1: GitHub Pages** (הכי פשוט!)

#### שלב 1: צור חשבון GitHub
1. לכי ל-[github.com](https://github.com)
2. לחצי "Sign Up" וצרי חשבון (חינם)
3. אשרי את המייל

#### שלב 2: צור Repository חדש
1. לחצי על ה-`+` בפינה הימנית למעלה
2. בחרי "New repository"
3. שם: `heroes-quest` (או כל שם שאת רוצה)
4. סמני ✅ "Public"
5. לחצי "Create repository"

#### שלב 3: העלי את הקבצים
1. לחצי "uploading an existing file"
2. גררי את **כל הקבצים** מהתיקייה
3. לחצי "Commit changes"

#### שלב 4: הפעל GitHub Pages
1. לכי ל-Settings (למעלה)
2. גללי ל-"Pages" בצד שמאל
3. תחת "Source" בחרי "main" branch
4. לחצי "Save"
5. **תוך דקה-שתיים** תקבלי קישור!

הקישור יהיה: `https://[שם-המשתמש].github.io/heroes-quest`

---

### **אופציה 2: Netlify** (מהיר ומומלץ!)

#### שלב 1: היכנסי לנטליפיי
1. לכי ל-[netlify.com](https://netlify.com)
2. לחצי "Sign Up" והיכנסי עם GitHub

#### שלב 2: פרוס את האתר
1. לחצי "Add new site" → "Deploy manually"
2. **גררי את כל התיקייה** לאזור ההעלאה
3. **זהו!** תוך שניות תקבלי קישור!

הקישור יהיה משהו כמו: `https://[שם-אקראי].netlify.app`

#### בונוס: שנה את השם
1. לחצי "Site settings"
2. "Change site name"
3. בחרי שם שאת רוצה: `heroes-quest.netlify.app`

---

### **אופציה 3: Vercel**

1. לכי ל-[vercel.com](https://vercel.com)
2. היכנסי עם GitHub
3. לחצי "Add New" → "Project"
4. העלי את התיקייה
5. לחצי "Deploy"

## 📱 איך הילדים משתמשים?

### **בפעם הראשונה:**
1. שלחי להם את הקישור (למשל: `https://yourname.github.io/heroes-quest`)
2. הם יפתחו אותו בדפדפן
3. יקפוץ להם הודעה: **"התקן את האפליקציה"**
4. לחצו "התקן עכשיו"
5. **זהו!** יש להם אייקון במסך הבית כמו אפליקציה רגילה! 🎉

### **מעכשיו:**
- פשוט לוחצים על האייקון
- עובד **גם ללא אינטרנט** (אחרי ההתקנה הראשונה)
- מתעדכן אוטומטית כשאת משנה משהו!
